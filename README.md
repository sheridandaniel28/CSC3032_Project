SmartEmotions is a web-based application designed to help autistic children under the age of 12 improve emotional recognition and understanding while also developing basic mathematical skills. The app combines interactive games, visual cues, and audio feedback to teach emotions such as happiness, sadness, anger, and surprise, alongside math activities like counting, addition, and subtraction.

Features:
Emotion Recognition: Identify emotions from images with audio prompts.
Math Module: Interactive exercises for counting, addition, and subtraction.
Progress Tracking: Parents and educators can view reports and set goals.
Gamification: Rewards and badges to keep children motivated.
Accessibility: Large buttons, clear icons, and audio instructions for ease of use.

Technology Stack:
Frontend: HTML, CSS
Backend: Java (JSP/Servlets), Apache Tomcat
Database: MySQL (managed via phpMyAdmin)
IDE: Eclipse
Version Control: GitLab
